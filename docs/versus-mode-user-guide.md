# 対戦モード ユーザーガイド

ラブカ・シミュレータの「対戦モード」の使い方。ソロ（1人分の練習）とは別に、**2人で盤面を共有**して遊ぶモード。

## モードの違い

| モード | ログイン | 相手 | 用途 |
|--------|----------|------|------|
| **ソロプレイ** | 不要 | なし（1人分の盤） | 効果・確率の練習 |
| **一人二役（localDual）** | 不要 | 同じ端末で自分が両者操作 | 相手盤に触れる効果のテスト・1人で通し練習 |
| **オンライン対戦（online）** | **ゲスト or Google** | 別端末の相手 | 2人でルーム接続して対戦 |

「一人二役」と「オンライン」は同じルール処理を共有する。オンラインは相手が別端末にいるだけで、
盤面の見え方・効果の挙動は一人二役と同一になるよう作られている。

## オンライン対戦のはじめ方

1. **メニュー → 対戦**（ヘッダの対戦ボタン、または対戦ロビーを開く）。
2. 「オンラインルーム」を開く。**ログインは自動**（ゲスト＝匿名ログイン）。Google ログインは
   登録デッキを同期したいときだけでOK。
3. **ホスト**: 「ルームを作成」→ 表示された**4文字のルームコード**を相手に伝える。
4. **ゲスト**: コードを入力して「コードで参加」。
5. 両者が**デッキを選んで「このデッキで準備完了」**。
6. ホストが「**対戦開始**」を押すと、両者の画面が対戦画面へ切り替わる。

> 注: ゲスト（匿名）ログインが無効というエラーが出る場合は、Firebase コンソールで
> Authentication → Sign-in method → 「匿名」を有効にする（管理者向け）。

## 対戦中の流れ

- **マリガン → 先攻通常ターン → 後攻通常ターン → …** と進む。手番は画面表示（自分のターン/相手のターン）で分かる。
- **ライブ**: ライブ枠に置く → set → perf → judgment → 成功なら成功ライブ置き場へ。
- **勝敗**（総合ルール 1.2.1）: 成功ライブ**3枚以上**かつ相手**2枚以下**で勝ち。**投了**も可能。

## 何が同期される？

| 同期される（公開盤） | 非公開 |
|----------------------|--------|
| ステージ・控え室・エネルギー・成功ライブ・ライブ枠、ターン、成功ライブ枚数、各種集計（スコア/ハート合計 等） | **手札**（効果で公開されるときだけ一時表示）、ライブの表面（ライブターン中は裏） |

## 相手の盤に触れる効果

「相手をウェイトにする」「相手の控え室を操作する」「相手に選ばせる」などの効果は、
**相手の端末で自動的に適用**される（ソロ用の入力ダイアログは出ない）。

- 発動側の画面には「**相手の操作を待っています…**」バナーが出る（経過秒・自動スキップまでの残り秒つき。**キャンセル不可**）。
- 相手が選ぶ効果では、相手の画面に選択ダイアログが出て、選んだ結果が自分の盤に反映される。
- 相手が**120秒**応答しない（切断など）と、その効果は**スキップ**されて進行する（総合ルール 1.3.2「可能な限り実行」）。

### 一部カードは手動指示
複雑な一部カード（山札の並べ替え等）は、オンラインでは**テキスト指示 → 相手が手動で盤を操作 →「完了」**で進む。
対象カードは [versus-online-known-gaps.md](./versus-online-known-gaps.md) を参照。

## 切断・再接続

- **リロード**しても、同じルームなら「**対戦画面へ**」で復帰できる。
- 相手が離脱しても、待ち効果はタイムアウトで解消され、進行は止まらない。
- **投了 → リマッチ**で次の対局を開始できる（前試合の盤面は持ち越さない）。
